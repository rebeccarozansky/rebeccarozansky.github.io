Faceblock

Seeing the world we live in today it's practically impossible to have anonimity. To solve this problem, I created a program that automatically blocks out faces in photos and videos with a blue box so that the face can not be reverse engineered.

How it Works

Using neural networks, my code identifies the location of each face in a picture or frame of a video and blocks it out given the coordinates of the faces spotted

ASL Live
alphabet translator

Using a dataset I found on Kaggle, I created a program which translates a live video feed from the webcam into a letter in sign language.


How it Works

I trained a neural network with Keras on a dataset found here. I then capture and process frames of the user's hand from the webcam with OpenCV and output the corresponding letter.

Election Maps

I created interactive election maps that display the results of the 2016 election by county. When you hover over a county on the map, it displays the winner of the election, the total votes, and the percent republican or democrat that county voted.

How it works

Using a dataset that can be found here, I used geopandas and plotly to create an interactive map.
